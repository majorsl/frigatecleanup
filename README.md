Frigate NVR is a fantastic piece of software, however even though you set a retention date, sometimes it leaves files behind. Maybe something crashed, maybe they were locked or in use.

Unfortauntely, Frigate won't go back and clean these up for you.  The Devs seem uninterested is resolving this, hence the script.

This script will remove orphaned files based on a retention time (default 11 days.)  It is a good idea to add an extra day for safety, so if your retention is 10 days, set it to
purge orphaned files on the 11th day.
